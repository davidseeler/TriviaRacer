Play now at http://triviaracer.me or https://triviaracer.herokuapp.com!

## Installation
**Node.js 10+ required**

Use the package manager [npm](https://www.npmjs.com) to install the following dependencies: 

```bash
npm install express
```
```bash
npm install socket.io
```
```bash
npm install node-fetch
```
